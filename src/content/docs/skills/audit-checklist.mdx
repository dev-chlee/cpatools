## 개요

감사 계획 단계에서 산업별·규모별 맞춤 체크리스트를 생성합니다.
한국채택국제회계기준(K-IFRS)과 감사기준서를 기반으로 합니다.

## 사용 예시

```bash
/audit-checklist --industry 제조업 --size 중견기업 --period 2025
```

위 명령 실행 시 해당 조건에 맞는 감사 체크리스트가 생성됩니다.
